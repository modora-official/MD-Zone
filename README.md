## ✦ System Overview

**MODORA** is a next-generation, zero-database Single-Page Application (SPA). It acts as a lightning-fast, centralized hub for downloading modified applications, games, and developer scripts.

By utilizing the **GitHub REST API** as a headless Content Management System (CMS), MODORA completely eliminates the need for backend servers. Every time a user loads the page, the application dynamically reads this repository and renders the UI instantly on the client side.

---

## 🚀 Core Technologies & Architecture

| Feature | Description |
| :--- | :--- |
| ⚡ **Serverless Data Fetching** | Real-time content generation via `api.github.com/repos/.../contents/`. No SQL, no databases required. |
| 🧠 **Intelligent Parsing** | Regex-powered engine that automatically cleans raw folder names (removes `v1.2`, `[MOD]`, underscores) for premium typography. |
| 🎨 **Glassmorphism UI** | OS-level dark mode aesthetics with fully customizable CSS variables (Colors, Fonts, Layouts). |
| 🔋 **Eco Mode** | Built-in battery saver toggle that disables heavy backdrop-filters and blur effects for low-end devices. |
| 🚀 **View Transitions API** | Native, silky-smooth animations when switching between List View and Grid View. |
